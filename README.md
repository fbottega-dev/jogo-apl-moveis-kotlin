# Jogo de Cliques - Android (Jetpack Compose)

Bem-vindo ao **Jogo de Cliques**! 🎮  
Este é um jogo simples desenvolvido em **Jetpack Compose** para Android. O objetivo do jogo é clicar na imagem exibida até alcançar um número de cliques definido aleatoriamente.

---

# Funcionalidades

# Tela Inicial
- Exibe opções para **iniciar o jogo** ou **desistir**.

# Durante o Jogo
- Mostra uma imagem que **muda conforme o progresso dos cliques**.
- O jogador pode **desistir a qualquer momento**.

#  Tela de Vitória
- Exibe uma **mensagem de vitória**.
- Opções para **jogar novamente** ou **sair do jogo**.

# Tela de Desistência
- Exibe uma **mensagem de desistência**.
- Opções para **começar um novo jogo** ou **sair do jogo**.

---

# Como executar

1. Abra o projeto no **Android Studio**.
2. Compile e execute em um **emulador ou dispositivo Android**.
3. Divirta-se clicando e tentando bater o número de cliques aleatório! 🎉

---

# Tecnologias usadas
- **Kotlin**
- **Jetpack Compose**
- **Android Studio**
